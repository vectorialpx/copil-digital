# Copil digital

Acest proiect poate fi folosit ca material de invatare dar este si o dovada a traseului datelor - as dori sa vad acest proiect in productie si este important de stiut ca nu vom salva nicio informatie personala, nici macar adresa de email.

## Despre ce e vorba?

Acest proiect este dedicat special parintilor, care vor sa stinga focul cu foc: o aplicatie care sa ne scape de alte aplicatii, in special social-media, dar si sa ofere informatii despre privacy si riscurile publicarii de informatii sensibile pe internet.

## Install

### Front end

```bash
git clone https://github.com/vectorialpx/copil-digital.git
cd copil-digital
cd client
npm install
npm run dev
```

Asta ar trebui sa rezulte in
```bash
 VITE v8.0.0  ready in ... ms
  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```
Iar `http://localhost:5173/` ar trebui sa fie disponibil in browser.
