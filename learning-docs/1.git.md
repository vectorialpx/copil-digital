# Despre GIT

Git este un instrument de versionare a codului - poti avea cod cu `versiune1` si cod cu `versiune2`, si poti sa le schimbi intre ele dintr-o simpla comanda 

Cu o simpla rulare de `git checkout versiune1`, toate fisierele tale se vor modifica in versiunea salvata ca `versiune1`.

## Scurta descriere

### Ce este?

Git este un program pe care trebuie sa-l [instalezi la tine pe laptop](https://git-scm.com/install/)

### La ce ne ajuta?

Codul pe care il scriem poate sa aiba versiuni (v1, v2 etc) si, foarte important, putem sa avem o vizibilitate a modificarilor de pe fiecare versiune (v1 modificarea 11, v1 modificarea 22, v2 modificarea 33, v2 modificarea 44 - unde 11, 22, 33, 44 sunt commit IDs).

Exemplu:
- Ai un fisier `test.js` care are textul `propozitie = 'Ana are mere'` - acesta este versiunea 1 a codului tau
- Sa zicem ca vrei sa modifici fisierul: `propozitie = 'Ana are mere dulci'`. Acest fisier poate sa fie in aceeasi versiune, dar ceea ce este foarte valoros este ca poti sa vezi ce modificare ai facut si cand - faptul ca ai adaugat "dulci", si la ce ora
- Sa zicem ca vrei sa schimbi codul `prop = 'Ana cu mere'`, dar nu esti sigur daca este o idee buna. Este simplu:
    - faci versiunea 2, pornind din versiunea 1
    - acum ai versiunea 2, care are acelasi continut ca versiunea 1 `propozitie = 'Ana are mere dulci'`
    - modifici codul, fiind pe versiunea 2, scrii `prop = 'Ana cu mere'` - acum ai doua versiuni de cod
- Poti sa schimbi oricand fisierul, sa fie pe v1 sau v2 dintr-o singura comanda de git, stiind numele de versiune - `git checkout versiune1` sau `git checkout versiune2`

### Poti sa-mi explici mai multe?

Nu, acesta este un material pe care va trebui sa-l studiezi singur(a) pentru ca are nevoie de ceva mai mult timp de invatare, poate sa fie folosit simplu dar se poate si complica foarte rapid.

Exista o multime de tutoriale si video-uri ale lui Git, pe care altii deja le-au discutat, nu ar avea rost sa fac eu un altul. Vezi materialele de mai jos si cauta pe Google: "learn git with easy steps", ca sa inveti in stilul tau.
